# VITALIS

Plataforma web de gestión nutricional inteligente para personas con diabetes.

VITALIS utiliza Inteligencia Artificial (Mistral) para generar planes de alimentación personalizados, registrar niveles de glucosa y ofrecer recetas adaptadas a las necesidades de cada usuario.

## Características

- **Menús semanales con IA** — Planes de 7 días (desayuno, almuerzo, cena y colaciones) generados automáticamente según preferencias, alergias y objetivos.
- **Control de glucosa** — Registro de niveles diarios con gráficos de evolución y reportes semanales.
- **Recetas paso a paso** — Instrucciones detalladas, ingredientes exactos y lista de compra organizada por categorías.
- **100% web** — Sin instalación. Funciona en cualquier dispositivo con navegador y acceso a internet.
